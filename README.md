## JOGO DE LUTA

Jogo multiplayer local

Comandos: 

- Movimentação player 1: <br>
A - Esquerda<br>
D - Direita<br>
W - Pula<br>
R - Ataque 1<br>
T - Ataque 2<br>

- Movimentação player 2:<br>
Left Arrow - Esquerda<br>
Right Arrow - Direita<br>
Up Arrow - Pula<br>
Num1 - Ataque 1<br>
Num2 - Ataque 2<br>
